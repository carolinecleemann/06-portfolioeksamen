02 GRUNDLÆGGENDE WEB

HTML

I begyndelsen af temaet blev vi introduceret til HTML (Hyper Text Markup Language), som er grundlaget for at lave hjemmesider. Vi lærte at bruge Visual Studio Code som editor og at oprette sider med korrekt HTML-struktur. Dette inkluderede brugen af tags og attributter, som giver elementer egenskaber. Eksempelvis bruges p til at definere afsnit og h1 til overskrifter. En vigtig del af undervisningen var at forstå “nesting,” altså den korrekte rækkefølge og lukning af HTML-tags. Vi fik også indsigt i, hvordan HTML-kode opmærker indhold, uden at definere dets udseende, og hvordan browsere som Chrome eller Firefox fortolker denne kode.

CSS

Efter introduktionen til HTML blev vi undervist i CSS (Cascading Style Sheets), som bruges til styling og layout. Vi lærte at oprette eksterne stylesheets, som linkes til HTML-filer, og hvordan CSS-selektorer anvendes til at style specifikke elementer. Dette inkluderer:

  • Element-selektorer (fx a, body).
  • Id-selektorer (fx #eksempel).
  • Class-selektorer (fx .container).
  • Pseudoselektorer (fx a:hover).

i blev introduceret til avancerede CSS-teknikker som shorthands, der gør det muligt at forenkle og samle flere deklarationer. Derudover lærte vi om viktigheden af en velorganiseret mappestruktur for at holde overblik over vores filer.

Grid og flexbox

Vi arbejdede med CSS Grid og Flexbox, som er layoutværktøjer til at strukturere sider på en responsiv og brugervenlig måde. Et grid består af en container (parent) og grid-elementer (child items). Disse værktøjer blev brugt i vores “mobile-first”-design, hvor vi fokuserede på at sikre brugervenlighed og design på tværs af skærmstørrelser.

opgaver

Gennem temaet udførte vi øvelser og opgaver som at designe et “computersite,” hvor vi anvendte designkonventioner, gestaltlove og wireframes. Vi skabte layoutdiagrammer og implementerede mobile-first designprincipper. Til studiestartsprøven demonstrerede vi vores viden om brugergrænseflader og designkonventioner. Temaet gav os en stærk forståelse for de grundlæggende redskaber i webudvikling og design. Vi arbejdede både teoretisk og praktisk med HTML, CSS og responsive designmetoder, hvilket sikrede et solidt fundament for fremtidige projekter.
